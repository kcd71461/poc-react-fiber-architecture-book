# React Fiber Architecture Ebook Rebuild Plan

## 목적

전자책 중심 UX로 재구성하고, React Fiber 내부 설계를 중고난도 독자에게 설명 가능한 콘텐츠를 단계적으로 확장한다.

## 독자 정의

- React 앱 개발은 익숙함
- 내부 Reconciler/Scheduler/Phase 설계는 더 깊게 알고 싶음
- 실무 디버깅과 성능 최적화에 바로 적용 가능한 지식 선호

## 릴리스 단계

### Phase 1 (현재)

- Nextra + MDX 기반 전자책 앱 골격 구축
- 01장(왜 Fiber가 필요했나) 우선 공개
- 리뷰 피드백 수집 체계 확립

### Phase 2

- 02~05장 공개
- 브라우저 API, 포인터 구조, Render/Commit, Double Buffering 심화

### Phase 3

- 06장 이후 실무 최적화/Profiler/RSC 확장
- 부록(실습/체크리스트) 정리

## 콘텐츠 규격

각 챕터는 아래 구조를 따른다.

1. 비유 소개
2. 문제 정의
3. 해결 방법
4. 기술적 구현 (TypeScript 의사 코드)
5. 브라우저 연동
6. 실습 예제 (코드리뷰형)
7. Source Map
8. 오해 바로잡기
9. 요약 체크리스트

## 품질 기준

- 기술 정확성: React 공식 설명과 충돌하지 않음
- 교육 밀도: 개념-구현-실무 연결이 단절되지 않음
- 검증 가능성: 장별 체크리스트로 학습 성취를 확인할 수 있음

## 01장 리뷰 요청 포인트

- 설명 난이도(너무 쉽거나 너무 어려운가)
- 비유와 실제 메커니즘 연결이 자연스러운가
- 실무 코드리뷰 문제로 전이 가능한가
- 추가로 보고 싶은 내부 모듈/함수는 무엇인가
